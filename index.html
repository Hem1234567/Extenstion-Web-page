Installation
Up in 6 steps
No account needed. Download, unzip, and load into Chrome in under a minute.

Download the ZIP

Click the download button above to get the extension package.

Extract the folder

Unzip the archive to a location you'll remember.

Open Extensions

Go to chrome://extensions in Chrome.

Enable Dev Mode

Toggle Developer mode in the top-right corner.

Load unpacked

Click "Load Unpacked" and select the extracted folder.

Pin to toolbar

Click the puzzle icon in Chrome and pin Hems for quick access.
