Step-by-Step Instructions
Because this extension is custom, you will need to load it in Developer Mode. It only takes 2 minutes.
1

Download the Extension

Click the blue "Download Extension ZIP" button at the top of this page. This will download a file named examly-ext-fixed--00 to your computer's Downloads folder.

2

Extract (Unzip) the downloaded file

Locate the downloaded ZIP file on your computer. Right-click on it and select "Extract All..." (on Windows) or double-click it (on Mac).

Choose a permanent location for the extracted folder (like your Documents or Desktop folder). Chrome needs this folder to remain on your computer for the extension to work.

Do not delete this folder later! If you delete or move the extracted folder after installation, the extension will break and stop working.

3

Open the Chrome Extensions Page

Open Google Chrome. Type chrome://extensions into your address bar at the top and press Enter.

Alternatively, click the three vertical dots (menu) in the top right corner of Chrome, select "Extensions", and then click "Manage Extensions".

4

Enable Developer Mode

In the top right corner of the Extensions page, you will see a toggle switch labeled "Developer mode". Click the toggle to turn it ON. The toggle should turn blue, and a new menu bar will appear at the top left of the page.

This setting is required to install custom extensions that are loaded directly from a folder on your computer.

5

Load the Unpacked Extension

Click the "Load unpacked" button that appeared in the top left corner of the Extensions page.

A file browser window will open. Navigate to the folder you extracted in Step 2. Select the folder (make sure you are selecting the folder that contains the manifest.json file inside it) and click "Select Folder".

6

Pin the Extension to your Toolbar

The Hems extension is now installed! To make it easy to access, click the Puzzle Piece icon (🧩) in the top right corner of Chrome, next to your address bar.

Find "Hems" in the dropdown list and click the Pin icon (📌) next to it. The Hems icon will now be permanently visible in your toolbar.

7

You're All Set!

Click the Hems icon in your toolbar to open the extension and start using it. You have successfully installed the extension.
